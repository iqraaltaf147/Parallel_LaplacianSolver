1 # Parallel Laplacian Solver using Random Walk
2 This repository will contain the implementation of the algorithm presented in
3 [this paper][paper]. The paper describes a random walk based method to solve an
4 important class of Laplacian Systems (Lx = b), called "one-sink" systems,
5 where exactly one of the coordinates of b is negative.
6 
7 ## Problem Statement
8 You are given an undirected positive weighted connected graph G = (V, E, w) with
9 adjacency matrix A<sub>uv</sub> = w<sub>uv</sub>. You are required to solve the
10 system of equations: Lx = b where L is the [Laplacian matrix][lapmat].
11 
12 ## Algorithm
13 The algorithm works by deriving the canonical solution from the stationary
14 state of the data collection process: Packets are generated at each node as an
15 independent Bernoulli process, transmitted to neighbors according to [stochastic
16 matrix][stomat] where P<sub>uv</sub> is directly propotional to w<sub>uv</sub>
17 and sunk at sink node only. Naturally, it consists of two phases: find parameter
18 &beta; such that DCP is ergodic and compute the stationary state, compute the
19 canonical solution by choosing an appropriate constant offset.
20 
21 ### Computing beta and stationary state
22 We have a lower limit for &beta;\* below which it's ergodic, so we binary
23 search this lower limit. Whenever it's not ergodic, there's one &eta; (queue
24 occupancy probability) which reaches 1, so we simply simulate the DCP at each
25 &beta; and check for this condition.
26 
27 ```
28 func estimateQueueOccupancyProbability(P, beta, J, T_mix, T_samp):
29  We generate and transmit for T_mix seconds, allowing it to reach
30  stationarity and then count the number of seconds queue is not empty for
31  T_samp time, to estimate occupancy probability
32 begin
33  t := 0
34  Q_t(u) := 0 for all nodes
35  cnt(u) := 0 for all nodes
36 
37  repeat
38  generate packets at all nodes according to Bernoulli(beta J_u)
39  transmit packets according to stochastic matrix, P
40  increment cnt if queue is not empty and t > t_mix for all nodes
41  t := t + 1
42  until t <= T_mix + T_samp
43 
44  report: cnt/T_samp as the estimate
45 end
46 
47 func computeStationaryState(P, b, t_hit):
48  We start with beta = 1 and reduce by half everytime we find that it's
49  non-ergodic
50 begin
51  T_mix := 64t_hit log(1/e1)
52  T_samp := 4logn/(k^2 e2^2)
53  J := -b/b_sink
54 
55  beta := 1
56  repeat
57  beta := beta/2
58  eta := estimateQueueOccupancyProbability(P, beta, J, T_mix, T_samp)
59  until max(eta) < 0.75(1 - e1 - e2)
60 
61  report: eta as the stationary state
62 end
63 ```
64 
65 ### Offset for canonical solution
66 The solution to Lx=b satisfies <<Lx, 1>> = 0 as &lambda;<sub>1</sub> = 0,
67 so we need to have an offset for stationary state such that this holds.
68 
69 ```
70 func computeCanonicalSolution(eta, beta, D, b):
71  Shifts the stationary state, eta by z*
72 begin
73  z* := -sum of eta(u)/d(u) for all u
74  sum_d := sum of d(u) for all u
75  x(u) := -b_sink(eta(u)/d(u) + z*d(u)/sum_d)/beta
76 
77  report: x as the solution to Lx=b
78 end
79 ```

85 ## IO format
86 The input will be of the following format
87 
88 ```
89 The first line in the file is a single integer n, denoting the number of nodes
90 The following n lines will have n space seperated real numbers, the Adjacency
91 matrix in row-major format
92 The final line will have n space seperated real numbers, the b in Lx=b
93 
94 A[i][i] = 0
95 A[i][j] = A[j][i] >= 0
96 b[n] = sum(b[1..n-1])
97 ```
98 
99 The output should be of the following format:
100 
101 ```
102 Print one and only line containing n space seperated real numbers, the solution
103 to Lx=b
104 ```
